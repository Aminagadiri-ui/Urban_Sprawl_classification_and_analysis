# Urban_Sprawl_classification_and_analysis
Multi-city remote sensing pipeline using Random Forest on Google Earth Engine to analyze urban sprawl from Landsat (30m) imagery. Sensor-agnostic (Landsat 5/7/8/9), it computes 8 urban expansion metrics across 6 time periods (2000–2025) for 4 climatically diverse cities, enabling reproducible long-term urban growth analysis.
